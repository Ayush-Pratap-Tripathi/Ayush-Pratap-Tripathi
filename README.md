## 🚀 Projects

### 🤖 ResReview — AI-Powered Resume Screening Platform
> Spring Boot · MongoDB · React.js · Hugging Face API · Apache Tika · JWT

An end-to-end resume analysis pipeline. Extracts text from uploaded resumes using Apache Tika, computes keyword overlap and semantic similarity via the Hugging Face Inference API, and delivers a match percentage with a rating. Secured with JWT authentication, environment-driven API key configuration, and a token-aware React frontend with Axios interceptors.

🔗 [View Repository](https://github.com/Ayush-Pratap-Tripathi/Condigence-LLP---Full-Stack-development-Intern/tree/main/ResReview)

### 💬 ChitChat — Real-Time Chat Application
> Node.js · Express.js · Socket.IO · MongoDB · React.js · Cloudinary · JWT

A presence-aware chat application with persistent messaging. Socket.IO powers real-time event broadcasting and online user tracking. Supports inline image sharing via Cloudinary (base64 → upload pipeline), JWT-secured routes with cookie-based auth, and a clean UX with toast notifications and input validation.

🔗 [View Repository](https://github.com/Ayush-Pratap-Tripathi/chitchat)

### ⭐ RateIt — Store Rating & Review Platform
> MySQL · Express.js · React.js · JWT

A role-based review platform with three user tiers — Admin, Store Owner, and User. Implements secure JWT auth (1-day expiry), connection pooling for production-grade MySQL performance, and full rating workflows. The frontend features centralized Axios instance with auto-token attachment, interactive dashboards per role, and client-side search with pagination-style loading.

🔗 [View Repository](https://github.com/Ayush-Pratap-Tripathi/RateIt)
